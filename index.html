Developer notes:

This is only the basic form of Pig Latin, more variations to be added in the future.

I'm working on making the layout work better on phones.

Known Bugs:

if you end a word with a digit (1-9) then the first letter + ay will still be put after it. This is working as intended please add a space after your digit if you wish it not to happen.

Apostrophies(') don't work properly yet, fixed soonTM
